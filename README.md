🔐 Password Strength Meter (Python)

📌 Description

This is a simple Python-based Password Strength Meter that checks whether a password is strong, medium, or weak.

It analyzes the password based on multiple security rules and also provides suggestions to improve it.

This project is useful for understanding basic cybersecurity concepts and string validation in Python.

🚀 Features


🔍 Checks password strength (Weak / Medium / Strong)

📏 Length validation (8+ characters)

🔠 Checks uppercase letters

🔡 Checks lowercase letters

🔢 Checks numbers

🔣 Checks special characters

💡 Provides improvement suggestions

🛠️ Requirements

Python 3.x

(No external libraries required)


▶️ How to Run

Clone the repository:

git clone https://github.com/your-username/password-strength-meter.git

Go to project folder:

cd password-strength-meter

Run the script:

python password_checker.py
